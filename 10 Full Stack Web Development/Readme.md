# Course Overview
This course provides an introduction to full stack web development, which involves three interwoven areas: front-end web development, back-end web development, and database support for the web application. A modern web application framework will be studied, and projects implementing a full web application was required.

This is the repository for the Full Stack Web Development course. This page will contain the coursework files that are completed during the course.  You will find sample code that was used to create the assignment and a brief description of the concepts or technology used to pass the assessment.
