#  Module 7 Assignment Overview

**Assignment #3**

Use the **world** MySQL database for this part of the exam. The world database is part of the MySQL download. It is also posted here for you to download.

Design SQL queries to do the following:
List Names of all countries in the Continent of North America

1. List Name, Continent, and Region for all countries whose Surface Area is more than or equal to 1,000,000
2. List Names of all countries whose GovernmentForm is Constitutional Monarchy
3. List country Names and their Capital (name) for all countries in the Continent of South America
4. List all fields for countries whose Population is more than 500,000,000
5. List all fields for countries whose GovernmentForm is Federal Republic
6. List sum of Population of each Continent (grouped by Continent)
7. List Names of countries whose Language is German
8. List all fields for countries whose Population is more than or equal to 200,000,000 and whose Surface Area is more than or equal to 500,000
9. List Names of countries whose Independence Year is NULL
10. List Names of all cities in the United Kingdom
11. List Names of all cities in the State (field name in the table is District) of Texas in the United States of America
12. List all fields for countries whose LifeExpectancy is between 50% and 70% (inclusive)
