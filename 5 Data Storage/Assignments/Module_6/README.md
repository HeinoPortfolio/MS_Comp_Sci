#  Module 6 Assignment Overview



Use the Sakila database to write SQL queries to accomplish the tasks described below. Execute the queries on the Sakila database. Submit your queries as a PDF document.
Questions that were answered in the assignment:

1. Add a new customer to the customer table with the following details:
  - First Name: "John"
  - Last Name: "Doe"
  - Email: "john.doe@example.com"
  - Store ID: 1
  - Active: 1

2. Insert a new record into the film table with these details:
  - Title: "Code Masters"
  - Description: "A movie about the world of coding."
  - Release Year: 2024
  - Language ID: 1
  - Rental Duration: 7 days
  - Rental Rate: 2.99
  - Replacement Cost: 19.99

3. Add a record in the actor table with:
  - First Name: "Emily"
  - Last Name: "Blunt"
4. Retrieve the first name, last name, and email of all active customers.

5. List the titles of films rented by the customer with ID 3.

6. Find all films with "Adventure" in their description. Display the title, description, and release year.

7. For all films with a rental rate less than 2.00, increase the rental rate by 0.50.

8. Delete the film titled "Code Masters" that you created earlier in the assignment.

9. Remove all records from the rental table where the rental date is more than 5 years old.

10. Retrieve the title and description of all films released in the year 2006.

11. Display the first_name, last_name, and email of all customers who have no rentals in the rental table.

12. Find the total number of films for each category. Display the category name and the film count.

13. Calculate the total revenue generated by each store. Display the store id and the total revenue.
