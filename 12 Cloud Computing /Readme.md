# Course Overview
Introductory course in cloud computing. The course provides the foundations for a solid understanding of the concepts, capabilities, and underlying technologies of cloud computing. This course enables one to effectively evaluate and exploit the business and technical benefits of cloud-based solutions. Hands-on experience was gained through projects exposing the participant to application programming interfaces (APIs) offered by cloud providers. The course taught the individual how to build and deploy powerful and efficient data-intensive applications in the cloud. 

This is the repository for the **Cloud Computing** course. This page will contain the coursework files that are completed during the course.  You will find sample code that was used to create the assignment and a brief description of the concepts or technology used to pass the assessment.
