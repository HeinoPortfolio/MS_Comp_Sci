# Course Overview
The course covers basic concepts and techniques of artificial intelligence, including intelligent agents, search, reasoning, knowledge representation, planning, and human-robot interaction. Examples from a variety of applications will be used to illustrate the use of these techniques as well as their current limitations.

This is the repository for the Artificial Intelligence course. This page will contain the coursework files that are completed during the course.  You will find sample code that was used to create the assignment and a brief description of the concepts or technology used to pass the assessment.
